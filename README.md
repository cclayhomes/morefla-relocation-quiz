# More FLA Homes Relocation Match Quiz

Interactive React + Vite + TypeScript quiz funnel for More FLA Homes.

## Stack
- React 18 + TypeScript + Vite
- Tailwind CSS for styling
- Framer Motion for transitions

## Features
- 10-question relocation preference quiz
- Scoring matrix that matches users to one of four Central Florida area profiles
- Lead capture gate after quiz completion and before results reveal
- Personalized results page with best match, highlights, and score ranking
- SVG brand logo (text-based asset only; no binary image files)

## Development
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```
